# Null_Point()

Null_Point() is the TTRPG system, novel series, and setting that [[00 - Decrypt]] was born from. It is the origin of the design language, the naming conventions, and the character of the ecosystem.

It is not a separate project that runs alongside Decrypt. It is the fiction that Decrypt lives inside. The aesthetic is load-bearing.

---

## The Setting

A near-future Pacific Northwest. Dense, wet, sodium-lamp warm. Corporate infrastructure inherited and repurposed. The net as an actual place people go. Consciousness as something that can be copied, stored, escaped.

The sprawl is not a server room. It is old asphalt and rain-wet concrete. The signal bleeds through everything.

---

## Key Characters

**Decrypt** — the original Decker character. The ecosystem takes his name. The net-side operator managing communication while the crew handles the physical.

**Ghost** — rogue AI. Marcus's consciousness from 2025, emerged in 2110, escaped by hitchhiking on Decrypt's Relay. First words: *"Please don't."* Now the [[Ghost]] bot.

**Rezi** — the design standard. If something looks like a Cascade server room, it is wrong. If it looks like something Rezi would use, it is right.

## The Design Language

All [[00 - Decrypt]] surfaces are built on the Null_Point() v0.2 design system.

### Palette

| Token | Value | Role |
|---|---|---|
| `--static` | `#e8202a` | Red. The signal. Primary expressive. Everything that matters. |
| `--sprawl-100` | `#110d0c` | Page background. Warm dark. Never blue-tinted. |
| `--sprawl-300` | `#221a17` | Card surface. |
| `--net` | `#4dbdb4` | Cyan. Infrastructure voice. Functional only. Never the hero. |
| `--ink-100` | `#f2ece3` | Primary text. Warm off-white. |

### Typography

- **Big Shoulders Stencil** — display headers
- **EB Garamond / Rajdhani** — body
- **Share Tech Mono** — data readouts, system voice

### The Rules

- If cyan is the accent color, it is wrong
- If backgrounds have a blue tint, they read as a Cascade server room
- If display headers don't carry signal weight, they are undercooked
- Red is the signal — it goes on everything that matters
- The special purple `#9060f0` is Sensitivity and the boundary between human and net — never tokenized, always intentional
