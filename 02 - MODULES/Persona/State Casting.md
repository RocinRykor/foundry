# State Casting

State Casting is the mechanic by which emotional state is converted into creative output — and creative output feeds back into emotional state.

It is not a metaphor. It is a description of how the system actually works for Steven specifically.

It is the answer [[Persona]] found when it looked at this person. Other people running Persona would find different answers.

---

## The Core Mechanic

Humans can transmit genuine emotional experience through acts of creation. The quality of the output is determined by how much real emotional investment went into it. Skill determines capacity. Emotion determines power.

The act of creating with genuine presence returns emotional value to the creator regardless of audience. This is the feedback loop.

```
Emotional State → Creative Act (with presence) → Output → Feedback → Pool Refill
```

**Presence is load-bearing.** Going through the motions produces weak or no feedback. The loop only closes if the mana spent was real.

---

## The Two Pools

Every person operates with two emotional pools. They accumulate passively through lived experience and deplete through expenditure in creative or destructive acts.

**Light Pool** — fed by joy, wonder, love, excitement, connection.
**Dark Pool** — fed by grief, rage, fear, shame, despair.

Neither is good or bad. They are fuel. The dark pool is not a problem to be eliminated — it is a resource to be spent.

**The key conversion:** Dark mana spent in genuine creative acts does not return as dark mana. What comes back is the emotional residue of having made something real. This is the exploit available to anyone carrying a dark surplus.

---

## The Lid Rule

**Before you open the vessel, define the lid.**

A cast session requires a completion condition defined in advance. Not a project goal. A specific, reachable endpoint:
- One scene written
- One mechanic documented
- One feature specced in a paragraph

Without a lid, the mana amplifies instead of converting. The loop stays open. This is wallowing — indistinguishable from productive casting in the early stages, but it deepens the pool instead of draining it.

**ADHD note:** Defining the lid requires executive function at the moment it is hardest to access — when the dark pool is highest and the pressure to cast is greatest. This is why pre-built menus matter.

## Implementation in [[00 - Decrypt]]

State Casting requires:

1. **State Reading** — [[Depot]] captures current pool state before initiating a cast
2. **Cast Preparation** — selecting a pre-defined completable task, confirming the lid
3. **Session Logging** — [[Depot]] records what was cast from, what was produced, feedback quality
4. **Pattern Recognition** — [[Persona]] identifies which creative acts generate the strongest conversion for this specific person over time
5. **Failure State Logging** — recording when a cast went into wallowing, without judgment, to identify conditions that produce failed casts
