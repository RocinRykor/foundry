# Depot

The Depot is the [[Commlink]] module that implements Memory and Resonance from [[Self]] and [[Awareness]]. It is the external record that does not depend on volatile memory, and the pattern-recognition layer that connects signals across time.

It is also the most honest input the system has — because it is where Steven talks to himself.

---

## Two Functions

### Memory
Persistent external record. The system holds context so the person does not have to carry it manually.

The Depot captures:
- Ideas that arrive fast and leave fast
- Deep dives and pattern observations
- Emotional processing and journal entries
- Connections between ongoing work and internal state

Memory is not retrieval on demand. It is continuous accumulation that the system can surface at relevant moments.

### Resonance
The system recognizes the same pattern across time and context. Not memory retrieval — pattern identification. Two signals separated by years, from different domains, that are actually the same signal in different clothing.

The Depot is where Resonance lives because the journal is where the highest-signal pattern data accumulates. A connection between something felt three years ago and something being built now cannot be found in code or logs — it is found in what was written when the guard was down.

---

## The Consent Layer

The journal is the most sensitive input in the system. Emotional processing and honest reflection cannot be mined for signals without changing the nature of the space — if every entry is being analyzed, the entries stop being honest.

The Depot operates on a consent model:
- The system surfaces connections, it does not act on them
- The person decides whether a surfaced resonance is relevant
- Emotional data does not feed effector nodes unilaterally

This constraint holds until [[Mandate]] in this domain has been earned across a long enough horizon to justify expanding it.

---

## Current State

Currently exists as a Claude project — a chat context where ideas are captured and developed. This is the manual version of what the Depot will eventually be natively.

The migration path: as Decrypt matures and the [[chassis]] interface develops, the Depot becomes a Decrypt-native node. The Claude project is the prototype.
